# Shortest Path

## Allgemein

```cypher
MATCH p = shortestPath(
	(a:User {id: "u1"})-[:FOLLOWS*..10]->(b:User {id: "u20"})  
)  
RETURN p;
```

## Typische Fragestellungen

> Wie viele Schritte benötigt u1, um u15 (AI-Influencer) über FOLLOWS-Beziehungen zu erreichen?

```cypher
MATCH p = shortestPath(
  (a:User {id:"u1"})-[:FOLLOWS*..10]->(b:User {id:"u15"})
)
RETURN length(p);
```

> Ist u15 oder u25 für u1 schneller erreichbar, und wie groß ist der Unterschied?

```cypher
MATCH p1 = shortestPath((:User {id:"u1"})-[:FOLLOWS*..10]->(:User {id:"u15"}))
MATCH p2 = shortestPath((:User {id:"u1"})-[:FOLLOWS*..10]->(:User {id:"u25"}))
RETURN length(p1) AS to_ai, length(p2) AS to_gaming;
``` 

> Wie schnell erreicht u1 andere FPGA-User innerhalb der eigenen Community?

```cypher
MATCH (start:User {id:"u1"}), (target:User)
WHERE target.group = "fpga" AND start <> target
MATCH p = shortestPath((start)-[:FOLLOWS*..10]->(target))
RETURN target.id, length(p)
ORDER BY length(p);
```

> Über welche User verläuft der kürzeste Weg von u1 zu u20?

```cypher
MATCH p = shortestPath(
  (a:User {id:"u1"})-[:FOLLOWS*..10]->(b:User {id:"u20"})
)
UNWIND nodes(p) AS n
RETURN n.id;
``` 

> Welche Community ist für u1 am nächsten?

```cypher
MATCH (start:User {id:"u1"}), (target:User)
WHERE start <> target
MATCH p = shortestPath((start)-[:FOLLOWS*..10]->(target))
RETURN target.group, avg(length(p)) AS avg_distance
ORDER BY avg_distance;
``` 

> Welche AI-User sind für u1 überhaupt erreichbar, und welche nicht?

```cypher
MATCH (start:User {id:"u1"}), (target:User)
WHERE target.group = "ai"
OPTIONAL MATCH p = shortestPath((start)-[:FOLLOWS*..10]->(target))
RETURN target.id, p IS NOT NULL AS reachable;
```

> Welcher AI-User ist von u1 am weitesten entfernt?

```cypher
MATCH (start:User {id:"u1"}), (target:User)
WHERE target.group = "ai"
MATCH p = shortestPath((start)-[:FOLLOWS*..10]->(target))
RETURN target.id, length(p) AS dist
ORDER BY dist DESC
LIMIT 1;
```

> Welche AI-User sind von u1 aus über die FOLLOWS-Beziehungen erreichbar, und wie viele Schritte benötigt u1 jeweils, um sie zu erreichen?

```cypher
MATCH (start:User {id:"u1"}), (target:User)
WHERE target.group = "ai"
MATCH p = shortestPath((start)-[:FOLLOWS*..10]->(target))
RETURN target.id, length(p)
ORDER BY length(p);
```
